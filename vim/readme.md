# VIM

Mapped Leader key to `\`. The abbrivation of the leader key will be noted as `L+`.

## Custom key bindings

| Key | Description |
|:---:|:------------|
|L+ev | Edit my .vimrc |
|L+sv | Source my .vimrc |
|L+/  | Clear highlighted searches |
|F2   | Switch to paste mode |
